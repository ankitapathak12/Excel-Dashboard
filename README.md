📊 Sales & Profit Dashboard (Excel)

📌 Overview
An interactive Excel dashboard designed to analyze sales and profit performance across different regions, product categories, and cities. This project transforms raw data into meaningful insights using dynamic visualizations and user-friendly controls.
________________________________________
✨ Key Highlights
•	Fully interactive dashboard with slicers 
•	Clean and modern UI design 
•	Dynamic charts that update based on filters 
•	Easy navigation with structured layout 
________________________________________
📊 Features
•	Region-wise filtering (Central, South, West) 
•	Category segmentation (Furniture, Office Supplies, Technology) 
•	Pie chart for sales distribution 
•	Bar chart comparing Sales vs Profit by city 
•	City-wise performance insights 
________________________________________
📈 Data Analysis Capabilities
•	Compare sales across regions 
•	Analyze category-level performance 
•	Identify top and underperforming cities 
•	Track profit trends and losses 
•	Support data-driven decision making
